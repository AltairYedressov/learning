**Backend Dockerfile** (`app/backend/Dockerfile`) -- rewrite to add non-root user per D-04:

```dockerfile
# -- Backend: Python FastAPI --------------------------------------------------
FROM python:3.12-slim AS backend

WORKDIR /app

# Create non-root user (UID/GID 1001 per D-02)
RUN groupadd -g 1001 appgroup && useradd -u 1001 -g appgroup -d /app appuser

COPY --chown=appuser:appgroup requirements.txt .
RUN pip install --no-cache-dir -r requirements.txt

COPY --chown=appuser:appgroup main.py .

# Prevent Python from writing .pyc files (readOnlyRootFilesystem compat)
ENV PYTHONDONTWRITEBYTECODE=1

USER appuser

EXPOSE 8000

CMD ["uvicorn", "main:app", "--host", "0.0.0.0", "--port", "8000"]
```

Key changes:
- Add `RUN groupadd -g 1001 appgroup && useradd -u 1001 -g appgroup -d /app appuser` BEFORE any COPY
- Change both COPY lines to use `--chown=appuser:appgroup`
- Add `ENV PYTHONDONTWRITEBYTECODE=1` to prevent __pycache__ writes on read-only FS
- Add `USER appuser` after all RUN commands (pip install needs root)
- Keep the `AS backend` stage name, WORKDIR, EXPOSE, CMD unchanged

**Frontend Dockerfile** (`app/frontend/Dockerfile`) -- rewrite to add non-root user per D-03:

```dockerfile
# -- Frontend: Node.js + Express ----------------------------------------------
FROM node:20-alpine

WORKDIR /app

# Create non-root user (UID/GID 1001 per D-02)
RUN addgroup -g 1001 appgroup && adduser -u 1001 -G appgroup -D appuser

COPY --chown=appuser:appgroup package.json package-lock.json* ./
RUN npm install --production

COPY --chown=appuser:appgroup . .

USER appuser

EXPOSE 3000

CMD ["node", "src/server.js"]
```

Key changes:
- Add `RUN addgroup -g 1001 appgroup && adduser -u 1001 -G appgroup -D appuser` (Alpine syntax per D-03)
- Change both COPY lines to use `--chown=appuser:appgroup`
- Add `USER appuser` after npm install (npm needs root for /usr/local writes)
- Keep WORKDIR, EXPOSE, CMD unchanged
