Output: `git rm -r app/backend app/frontend` staged, README updated.

Honors D-07 (retirement), D-12 (no cluster-side verification — that is Phase 4).

Note: this is reversible via `git revert` if needed before merge. After merge, the trees remain recoverable from history for at least the repo's retention window.
</objective>
